# 贪吃蛇小游戏 (Snake Game)

这是一个使用 Python 和 Pygame 库开发的经典贪吃蛇小游戏。

## 核心功能

- **流畅控制**：使用键盘方向键控制蛇的移动。
- **得分系统**：吃掉食物得分（10分/个），并实时记录最高分。
- **游戏状态管理**：包含开始、暂停、游戏结束和重新开始功能。
- **音效支持**：支持背景音乐和动作音效（需手动添加音频文件）。
- **流畅运行**：帧率控制在 12 FPS，保证经典的游戏手感。

## 环境要求

- Python 3.x
- Pygame 库

## 安装步骤

1. 确保已安装 Python。
2. 安装依赖库：
   ```bash
   pip install -r requirements.txt
   ```

## 运行游戏

在终端或命令行中运行：
```bash
python snake_game.py
```

## 操作说明

- **方向键 (↑ ↓ ← →)**：控制蛇的移动方向。
- **P 键**：暂停/继续游戏。
- **R 键**：游戏结束后重新开始。
- **退出**：点击窗口关闭按钮。

## 自定义音效 (可选)

如果你想要开启音效，请将以下文件放在游戏根目录下：
- `eat.wav`: 吃掉食物的音效。
- `game_over.wav`: 游戏结束的音效。
- `background.mp3`: 背景音乐。

## 代码结构

- `Snake`: 处理蛇的逻辑（移动、增长、碰撞检测）。
- `Food`: 处理食物的随机生成。
- `Game`: 游戏主类，管理循环、渲染、状态和 HUD。
- `high_score.txt`: 自动生成的最高分记录文件。
